Kamu adalah developer Python yang akan memperbaiki bot scalping MT5 berbasis PySide6 di Windows.
Bot sudah punya GUI, analisa, dan eksekusi order, tapi saat dijalankan muncul error berikut:

1) UnicodeEncodeError pada logging:
UnicodeEncodeError: 'charmap' codec can't encode character '\u2705'

scss
Copy
Edit
- Ini terjadi karena di Windows console (CP1252) tidak bisa encode emoji (✅, ❌).
- SOLUSI: ganti log message agar tidak mengandung emoji atau set logger encoding UTF-8 yang aman di Windows.
- Kalau ingin tetap pakai emoji, gunakan:
  ```python
  import sys
  sys.stdout.reconfigure(encoding='utf-8')
  sys.stderr.reconfigure(encoding='utf-8')
  ```
  Pastikan ini dilakukan sebelum logging.basicConfig().
- Atau gunakan emoji hanya di GUI, sedangkan log file/console pakai teks biasa.

2) AttributeError: QTextEdit tidak punya method setMaximumBlockCount:
'PySide6.QtWidgets.QTextEdit' object has no attribute 'setMaximumBlockCount'

markdown
Copy
Edit
- Method `setMaximumBlockCount()` hanya ada di QPlainTextEdit, bukan QTextEdit.
- SOLUSI: Ganti QTextEdit menjadi QPlainTextEdit untuk log viewer, atau hapus pemanggilan setMaximumBlockCount jika tetap ingin QTextEdit.
- Rekomendasi: gunakan QPlainTextEdit untuk log karena performa lebih baik dan mendukung `setMaximumBlockCount()`, sehingga log tidak membengkak.
